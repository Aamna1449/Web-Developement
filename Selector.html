CSS Selector

A selector is used to select an html element for styling.

  1. Element Selector

    2. It is used to select an element based off the tagname.

    for example: h2{ color:violet; }

  2. ID Selector

    3. It is used select an element with a given id. It is denoted by "#".

    for example: #id{ color:maroon; }

  3. Class Selector

    4. It is used select an element with a given class. It is denoted by "."

    for example: .class{ color: burlywood; }
